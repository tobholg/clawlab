# Security Policy

If you believe you have found a security vulnerability in ClawLab, do not open a public issue with exploit details.

Please use GitHub private vulnerability reporting for the repository if it is enabled. If it is not enabled, contact the maintainer privately through GitHub before disclosing details publicly.

Include:

- A short description of the issue
- Reproduction steps or a proof of concept
- Impact assessment
- Any suggested mitigation

I will aim to acknowledge reports promptly and coordinate a fix before public disclosure.
